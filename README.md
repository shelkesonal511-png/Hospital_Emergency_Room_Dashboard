## 📌 Overview
The **Hospital Emergency Room Dashboard** is an end-to-end Excel project designed to analyze emergency room operations through interactive visualizations.

It provides a **monthly report of patient data**, helping stakeholders monitor hospital performance, improve efficiency, and enhance patient experience.

---

## 🎯 Objectives
- Track patient volume and flow  
- Monitor average wait times  
- Evaluate patient satisfaction  
- Analyze admission trends  
- Identify delays in patient attendance  
- Understand demographic and departmental distribution
